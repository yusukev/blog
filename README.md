# yusukev.com

Personal blog built with [Astro](https://astro.build) and deployed to [Cloudflare Workers](https://workers.cloudflare.com).

## Setup

```sh
pnpm install
```

## Development

```sh
pnpm dev
```

## Build & Deploy

```sh
pnpm build
```

Deployment to Cloudflare is automated via GitHub Actions on push to `main`.
